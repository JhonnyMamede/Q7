Fazer um programa em C para ler 4 números inteiros e calcular a média ponderada
da seguinte forma: o 1 número vai ter peso 3, o segundo vai ter peso 2, o terceiro vai ter peso
1 e o quarto vai ter peso 5. Deve-se imprimir os números digitados pelo usuário e na linha abaixo,
a média ponderada.
Para calcular a média ponderada, é preciso:
Multiplicar cada valor do conjunto de dados pelo seu peso;
Somar os resultados das multiplicações;
Dividir o resultado obtido pela soma dos pesos utilizados.
